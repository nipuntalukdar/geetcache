geetcache
=========
A Scalable, highly-available persistent distributed cache
---------------------------------------------------------

Geetcache is a scalable, distributed cache with persistence. Partitions are replicated for high
availability.

**Staus**

---

Completed features:

* simple key create, delete
* list put, get, pop, delete
* counters
* hyperloglogs
